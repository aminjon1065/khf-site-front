# Handoff: Редизайн портала КЧС Таджикистана (GovTech)

## Обзор
Полный редизайн официального портала Комитета по чрезвычайным ситуациям и гражданской обороне Республики Таджикистан (khf.tj). Цель — превратить новостной госсайт в современную платформу безопасности населения: статус угроз за 5 секунд, инструкции, карта рисков, официальная информация.

## О файлах дизайна
Файлы `*.dc.html` в этом пакете — **дизайн-референсы, созданные в HTML** (прототипы, показывающие внешний вид и поведение). Это НЕ production-код. Задача — **воссоздать эти макеты в существующем Next.js/Inertia-проекте** (репозиторий `khf`: Laravel + Inertia + React + Tailwind CSS 4 + lucide-react), используя его паттерны: существующие маршруты, CMS-модели, локализацию (`lang/ru|tj/ui.php`), хук `use-public-navigation`, компоненты в `resources/js/components/Public/`. Ничего из бизнес-логики, API и CMS не переписывать — только представление.

Каждый `.dc.html` открывается в браузере. Разметка внутри `<x-dc>…</x-dc>` — обычный HTML с inline-стилями (шаблон), плюс класс `Component` (логика, аналог React-компонента: `renderVals()` возвращает данные/обработчики для `{{ holes }}`). Переносите 1:1 в JSX/TSX-компоненты.

## Точность
**High-fidelity**: цвета, типографика, отступы, состояния и тексты — финальные. Воссоздавать пиксель-в-пиксель, но через Tailwind-токены/CSS-переменные проекта (см. «Дизайн-токены»).

## Дизайн-система
База — «Industry»: светлый технический фон, стальной синий акцент, Barlow Condensed (заголовки, uppercase) / Barlow (текст), квадратные углы, тонкие hairline-рамки. Файл токенов: `_ds/industry-.../styles.css` (в пакете). Иконки — Lucide, stroke-width 1.5 (в проекте уже есть lucide-react).

ВАЖНО: класс `.blueprint` исторически рисовал крестики по углам — по решению заказчика они **отключены** (см. CSS в Header.dc.html: `.blueprint .corner{display:none}`, рамка = 1px divider + shadow-sm). Реализуйте карточки просто как квадратные блоки с тонкой рамкой и лёгкой тенью.

## Дизайн-токены
Шрифты: `--font-heading: "Barlow Condensed", sans-serif` (600), `--font-body: "Barlow", sans-serif` (Google Fonts, кириллица).
База (светлая): `--color-bg #f2f2f3`, `--color-text #1d1f20`, accent `#5980a6` (ramp 100–900; accent-700 для текстовых ссылок), divider = `color-mix(text 16%, transparent)`.
Тёмная тема (`[data-theme="dark"]`, переключатель в служебной панели, хранится в `localStorage['kchs-theme']`): bg `#15181b`, surface `#1e2226`, text `#e4e7ea`, accent `#8fb2d4/#a4c2de/#bdd3e8`.
Семантическая шкала опасности (светлая / тёмная):
- success `#2f7d5a` bg `#e0efe7` / `#6dbb96` bg `#1c2f26` — штатно, завершено
- info `#416180` bg `#e3edf6` / `#8fb2d4` bg `#20303f` — информация
- warning `#96660f` bg `#f5e9d2` / `#d9a744` bg `#332a17` — предупреждение (оранжевый уровень)
- danger `#ac4d10` bg `#f7e4d3` / `#e0824a` — опасность
- critical `#a8231b` (hover `#8c1d16`) bg `#f8ddda` / `#e26a60` bg `#3a1d1a` — критично, кнопка 112
Правило: красный ТОЛЬКО для опасности и 112. Уровень всегда дублируется текстом/значком, не только цветом.
Сетка: контент max-width 1160px, поля 24px (16px mobile), секции через 52px, заголовки секций с нижней линией-divider и нумерацией «01 / Инструкции».

## Header (компонент)
Три яруса: (1) служебная панель — флаг+герб, ссылка «Государственные символы», переключатель темы (луна/солнце), сегмент языков РУ/ТҶ/EN (для отсутствующих переводов — информационная полоска-нотис); (2) бренд-строка — лого КЧС (assets/logo-kchs-ru|tj|en.webp), название в 2 строки uppercase + подстрока «при Правительстве…», телефон доверия, красная кнопка 112 (`tel:112`); (3) навигация — Главная, О нас (дропдаун: Руководство, Структура, Гос. символы), Новости, Безопасность, Карта рисков, Документы, Контакты; справа кнопка-аутлайн «Приложение SOS» и поиск. Активный пункт — accent-подчёркивание 2px.

## Мобильная версия (≤920px)
- Шапка одной строкой без переноса: лого+название (подстрока скрыта) слева, 112 компактная, бургер 46×46 справа. Полоса навигации скрыта.
- Бургер открывает **выдвижную панель справа** (width min(340px,90vw), затемнённый backdrop, закрытие крестиком/по фону): поиск, пункты 48px, секция «О нас» с подпунктами, «Приложение SOS», внизу — прижатая кнопка «112 — экстренный вызов» и телефон доверия.
- Все многоколоночные сетки → 1 колонка; быстрые действия и показатели → 2 колонки (→1 при ≤560px); слайдер: фото сверху; таблица документов — горизонтальный скролл; футер 2→1 колонки. Touch-targets ≥44px. В Next.js реализуйте нормальными Tailwind-брейкпоинтами (md:, lg:), а не `!important`-хаком прототипа.

## Взаимодействия
- Слайдер: transform translateX, transition .5s; точки-полоски 22×4px; `prefers-reduced-motion` отключает автопрокрутку.
- Критический баннер: точка-пульс 2.4s ease-in-out (opacity 1→.55), отключается при reduced-motion. Никакого мигания.
- Фильтры (новости/документы/объявления/карта) — кнопки-переключатели: активная = accent-заливка, `aria-pressed`.
- Дропдаун «О нас»: `aria-expanded/haspopup`, закрытие по клику вне.
- Формы: label всегда (не placeholder), ошибка объясняет как исправить, согласие на обработку ПД обязательно.
- Доступность: skip-to-content, focus-visible 2px accent, семантические заголовки, aria-current="page", карта дублируется списком.

## Состояние
- `alertState: 'calm'|'warning'|'critical'` — глобальный режим (из CMS/API уведомлений).
- Тема: localStorage `kchs-theme`, атрибут `data-theme="dark"` на `<html>`.
- Регионы карты: `{key, name, level: none|info|warning|danger|critical, count, statusText}`.
